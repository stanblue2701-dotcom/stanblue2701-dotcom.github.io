Notes

Working principles

Agents need instrumentation.

Autonomy is only useful when developers can inspect the trajectory, reproduce failures, and compare changes over time.

Memory should be typed.

Preferences, task traces, retrieved facts, and operating procedures should not live in one undifferentiated context blob.

Human control is a design surface.

The best approval flow is specific, fast, and auditable; it does not ask users to supervise every token.
